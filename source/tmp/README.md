# Solution format:

`ruuuulldDuurrddLUdrddlUruulllddRUluR`

Uppercase: push box
Lowercase: player move

# Level format:

```
#########
#       #
#@$ $ $ #
#### #  #
#  #$#$##
# ...  #
# ..*# #
##     #
 #######
```


- `@` player
- `$` box
- `.` goal
- `#` wall
- `*` box on goal
- `+` player on goal
